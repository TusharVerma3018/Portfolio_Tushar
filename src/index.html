Global Superstore Sales Dashboard Using Power Bi

This project aims to create a dynamic sales dashboard using Power BI for Global Superstore, focusing on two key aspects: analyzing sales data for the top three customers and aggregating total sales by region worldwide. Cricket World Cup 2022 Player Analysis Using Power Bi

Vrinda Store Annual Report Using MS Excel

This project aims to create an insightful annual report dashboard using MS Excel for VRINDA Store, focusing on key performance metrics and trends. The analysis of top 5 states by sales enables the store to allocate resources strategically and target high-potential markets effectively. Additionally, comparing sales across different age groups and genders provides valuable insights for tailoring marketing strategies. Moreover, analyzing sales performance against order volumes helps VRINDA Store assess its revenue generation efficiency and optimize inventory management. Through MS Excel's robust capabilities, this dashboard empowers VRINDA Store to make informed decisions and drive business growth in 2023 and beyond.
